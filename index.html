เกี่ยวกับฉัน

สวัสดีค่ะ ดิฉันเป็นโปรแกรมเมอร์ที่มีความหลงใหลในการสร้างซอฟต์แวร์ที่มีประสิทธิภาพและสวยงาม ดิฉันมีประสบการณ์ในการพัฒนาเว็บแอปพลิเคชันและโมบายแอปพลิเคชัน มีความมุ่งมั่นในการพัฒนาแอปพลิเคชันเว็บที่มีความทันสมัยและตอบสนองต่อการใช้งาน โดยใช้เทคโนโลยีด้าน Front-End ข้าพเจ้ามีประสบการณ์จริงในการออกแบบอินเทอร์เฟซที่เป็นมิตรต่อผู้ใช้ และปรับปรุงแอปพลิเคชันให้สามารถใช้งานได้อย่างมีประสิทธิภาพบนอุปกรณ์หลากหลายประเภท ข้าพเจ้ายินดีที่จะพัฒนาความรู้และทักษะอย่างต่อเนื่อง พร้อมที่จะมีส่วนร่วมในการพัฒนาโครงการนวัตกรรมต่างๆ ในตำแหน่ง Front-End Developer และ Programmer

ผลงาน

Waste Management Application (JS, HTML, PHP, CSS)

แอปพลิเคชันการจัดการขยะร่วมมือกับฝ่ายพัฒนาสิ่งแวดล้อมในการสแกนบาร์โค้ดของภาชนะเครื่องดื่ม เพื่อให้คำแนะนำในการจัดการขยะ และมีจุดทิ้งขยะที่กำหนดไว้ในพื้นที่มหาวิทยาลัยพร้อมให้ความรู้ต่าง ๆ เกี่ยวกับขยะ

Flower Shop (HTML, CSS, MySQL, PHP)

ร้านดอกไม้ที่ให้ผู้ใช้ซื้อดอกไม้หลากหลายชนิด มีการผสมรวมกับ MySQL เพื่อการจัดการข้อมูล และทำงานได้สะดวกต่อทั้งผู้ใช้และแอดมินจัดการข้อมูล

Hallowee Game (Java)

เป็นเกมที่ได้รับแรงบันดาลใจจากการเล่นแบบ Mario ที่ผู้เล่นต้องทำการต่อสู้กับมอนสเตอร์ต่าง ๆ ตลอดทางเพื่อเดินทางไปสู่เส้นชัยสุดท้าย

Front-end Developer - Intern (React, SCSS)

โปรเจกต์นี้เป็นสไลด์โชว์ภาพที่สามารถควบคุมได้ด้วยการปัด (swipe) และปุ่มนำทาง (แบบทดสอบการสมัครเข้าฝึกงาน)
